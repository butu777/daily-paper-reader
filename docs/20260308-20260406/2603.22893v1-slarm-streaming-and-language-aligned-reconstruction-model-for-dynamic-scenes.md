## 摘要
我们提出了 SLARM，这是一种统一了动态场景重建、语义理解和实时流式推理的前馈模型。SLARM 通过高阶运动建模捕捉复杂的非均匀运动，且仅在可微渲染上进行训练，无需任何光流监督。此外，SLARM 从 LSeg 中蒸馏语义特征，以获得语言对齐的表示。这一设计实现了通过自然语言进行语义查询，且语义与几何之间的紧密耦合进一步增强了动态重建的准确性和鲁棒性。此外，SLARM 使用基于窗口的因果注意力机制处理图像序列，在不累积内存开销的情况下实现了稳定、低延迟的流式推理。在这一统一框架内，SLARM 在动态估计、渲染质量和场景解析方面达到了最先进的水平，相比现有方法，其运动准确度提升了 21%，重建 PSNR 提升了 1.6 dB，分割 mIoU 提升了 20%。

## Abstract
We propose SLARM, a feed-forward model that unifies dynamic scene reconstruction, semantic understanding, and real-time streaming inference. SLARM captures complex, non-uniform motion through higher-order motion modeling, trained solely on differentiable renderings without any flow supervision. Besides, SLARM distills semantic features from LSeg to obtain language-aligned representations. This design enables semantic querying via natural language, and the tight coupling between semantics and geometry further enhances the accuracy and robustness of dynamic reconstruction. Moreover, SLARM processes image sequences using window-based causal attention, achieving stable, low-latency streaming inference without accumulating memory cost. Within this unified framework, SLARM achieves state-of-the-art results in dynamic estimation, rendering quality, and scene parsing, improving motion accuracy by 21%, reconstruction PSNR by 1.6 dB, and segmentation mIoU by 20% over existing methods.

### 2. 核心方法论
SLARM 提出了一个统一的 4D 高斯推理框架，其核心技术包括：
*   **高阶运动建模（Higher-Order Motion Modeling）**：
    *   不使用简单的匀速假设，而是采用泰勒展开（Taylor Expansion）建模位移函数。
    *   网络预测 0 到 $L-1$ 阶的运动系数（速度、加速度、加加速度/Jerk），通过公式 $\Gamma(\Delta t) = \sum_{l=0}^{L-1} \frac{m_l \cdot (\Delta t)^{l+1}}{(l+1)!}$ 计算位移。实验中采用 $L=3$（三阶）。
*   **语言对齐的 4D 语义蒸馏**：
    *   从 2D 基础模型 **LSeg** 中蒸馏语义特征到 3D 高斯原语中。
    *   每个高斯球携带一个高维语义向量，通过 alpha-blending 渲染出语义特征图，并与 LSeg 的输出进行 MSE 损失对齐。
    *   支持通过 CLIP 文本嵌入进行自然语言查询（如查询“行人”）。
*   **流式推理架构（Streaming Inference）**：
    *   采用基于窗口的因果注意力机制（Window-based Causal Attention），只关注当前和过去的帧。
    *   通过传播紧凑的隐藏状态实现增量更新，保证了恒定的低延迟和内存消耗，避免了随序列增长而导致的显存爆炸。
*   **自监督学习**：仅通过渲染损失（RGB、深度、感知损失 LPIPS）进行训练，无需真实的光流（Scene Flow）标注。

### 6. 主要结论与发现
*   **性能领先**：在动态区域重建上，PSNR 提升了 1.5 dB 以上；在场景流估计上，误差显著低于 STORM。
*   **语义增强**：通过 3D 几何先验的引入，SLARM 的语义分割效果甚至超过了其教师模型 LSeg 和其他强力 2D 基准（mIoU 提升 20%）。
*   **流式优势**：SLARM-W（窗口模式）在保持高精度的同时，实现了线性的推理时间和稳定的内存占用，适合长序列部署。

### 7. 优点与亮点
*   **统一性**：在一个前馈网络中同时解决了重建、运动和语义三个任务，实现了任务间的互补。
*   **非线性运动捕捉**：高阶泰勒展开有效解决了动态场景中非匀速运动导致的“重影”或“拉伸”问题。
*   **部署友好**：流式架构解决了 3DGS 在处理长视频时的显存瓶颈，对自动驾驶和具身智能应用具有重要意义。
*   **零样本查询**：语言对齐特性使其能够直接与 LLM 交互，无需重新训练即可识别新类别。

### 8. 不足与局限
*   **位姿依赖**：模型仍然依赖于准确的相机位姿（Camera Poses），在位姿估计不准的情况下性能会下降。

*   **计算开销**：虽然是前馈模型，但 64 台 NPU 的训练成本依然较高，对于普通研究者来说复现门槛较高。
*   **场景覆盖**：主要集中在自动驾驶（室外）场景，对于室内极近距离或超大规模室内场景的泛化能力尚待验证。
