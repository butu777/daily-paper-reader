## 摘要
我们提出了 F3DGS，这是一个用于去中心化多智能体 3D 重建的联邦 3D 高斯泼溅（3DGS）框架。现有的 3DGS 流水线假设可以集中访问所有观测数据，这限制了它们在分布式机器人场景中的适用性，因为在这些场景中，智能体独立运行，且集中式数据聚合可能受到限制。直接将集中式训练扩展到多智能体系统会引入通信开销和几何不一致性。F3DGS 首先通过配准来自多个客户端的局部合并 LiDAR 点云来构建共享几何支架，以初始化全局 3DGS 模型。在联邦优化过程中，高斯位置被固定以保持几何对齐，而每个客户端仅更新与外观相关的属性，包括协方差、不透明度和球谐系数。服务器使用可见性感知聚合来汇总这些更新，根据每个客户端观察每个高斯的频率来加权其贡献，从而解决了多智能体探索中固有的部分可观测性挑战。为了评估去中心化重建，我们收集了一个包含同步 LiDAR、RGB 和 IMU 测量值的多序列室内数据集。实验表明，F3DGS 在实现跨智能体分布式优化的同时，达到了与集中式训练相当的重建质量。该数据集、开发工具包和源代码将公开发布。

## Abstract
We present F3DGS, a federated 3D Gaussian Splatting framework for decentralized multi-agent 3D reconstruction. Existing 3DGS pipelines assume centralized access to all observations, which limits their applicability in distributed robotic settings where agents operate independently, and centralized data aggregation may be restricted. Directly extending centralized training to multi-agent systems introduces communication overhead and geometric inconsistency. F3DGS first constructs a shared geometric scaffold by registering locally merged LiDAR point clouds from multiple clients to initialize a global 3DGS model. During federated optimization, Gaussian positions are fixed to preserve geometric alignment, while each client updates only appearance-related attributes, including covariance, opacity, and spherical harmonic coefficients. The server aggregates these updates using visibility-aware aggregation, weighting each client's contribution by how frequently it observed each Gaussian, resolving the partial-observability challenge inherent to multi-agent exploration. To evaluate decentralized reconstruction, we collect a multi-sequence indoor dataset with synchronized LiDAR, RGB, and IMU measurements. Experiments show that F3DGS achieves reconstruction quality comparable to centralized training while enabling distributed optimization across agents. The dataset, development kit, and source code will be publicly released.

### 2. 方法论：核心思想与关键技术
F3DGS 提出了一种**几何-外观解耦**的联邦学习框架，主要包含以下三个阶段：
*   **度量尺度位姿构建（Metric-Scale Pose Construction）**：
    *   利用 LiDAR SLAM（如 KISS-SLAM）或单目深度估计（如 Depth Anything）获取各智能体的局部轨迹。
    *   通过 Umeyama 算法将局部轨迹对齐到全局 LiDAR 锚点地图，并使用指数衰减的 SE(3) 插值进行边界平滑，确保全局位姿的一致性。
*   **共享几何支架初始化（Global Initialization）**：
    *   在联邦优化开始前，利用配准后的 LiDAR 点云初始化一个全局共享的高斯基元集合（约 60 万个点）。
    *   **关键约束**：在后续的联邦训练中，**固定（Freeze）高斯中心位置 ($\mu_i$)**，仅允许客户端优化外观属性（缩放、旋转、不透明度和球谐系数）。这从根本上消除了几何漂移。
*   **可见性感知联邦聚合（Visibility-Aware Aggregation）**：
    *   **局部优化**：每个客户端在本地数据上进行 $T$ 步优化，并记录每个高斯基元被渲染的频率（可见性计数 $v_{k,i}$）。
    *   **全局聚合**：服务器根据可见性计数对客户端的更新进行加权聚合。公式为：$\alpha_{k,i} = \frac{v_{k,i}}{\sum v_{j,i} + \epsilon}$。这种方式解决了多智能体探索中的“部分可观测性”问题，即只让看到过该区域的智能体贡献权重。

*   **性能表现**：F3DGS 能够达到与中心化训练相近的重建质量，全局 PSNR 在多数序列中保持在 20-24 dB 之间。
*   **几何固定的有效性**：实验证明，固定几何支架并不会阻碍局部重建质量，反而极大地增强了联邦聚合的稳定性。

*   **解耦设计**：巧妙地通过固定几何位置解决了联邦学习中 3DGS 的参数对齐难题。
*   **可见性加权**：引入可见性感知聚合，符合多智能体协作中“谁看到谁负责”的直觉，有效处理了非独立同分布（Non-IID）的观测数据。
*   **实用性强**：不依赖原始图像交换，极大地降低了通信带宽需求，保护了数据隐私。

### 8. 不足与局限
*   **几何修正能力缺失**：由于高斯中心被固定，模型无法纠正初始化阶段存在的残余位姿误差或几何缺陷。

*   **场景单一**：目前仅在室内走廊验证，对于动态物体、大规模室外环境的鲁棒性有待考察。
*   **基准对比不足**：缺乏与最先进的非联邦分布式 3D 重建方法的横向对比。
